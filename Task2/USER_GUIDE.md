# Task 2 — User Guide (Heap + Heap Sort)

## Requirements
- Python 3.10+ recommended
- No third‑party packages required

## How to run the demo
From the repo root:

```bash
python -m task2.demo
```

## Files
- `heap.py` — array-backed binary heap (self-implemented)
- `heap_sort.py` — heap sort using our heap
- `demo.py` — quick demonstration

## Notes
- This task intentionally avoids using Python's built-in `heapq`.
